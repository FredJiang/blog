|                  命令                 |            作用            |
|---------------------------------------|----------------------------|
| `git reflog --date=local branch_name` | 显示分支创建的时间         |
| `git rm    --cached <file>`           | 删除 .gitignore 中的文件   |
| `git rm -r --cached <folder>`         | 删除 .gitignore 中的文件夹 |
|                                       |                            |
